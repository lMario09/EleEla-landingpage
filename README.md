# Ele&amp;Ela — Landing Page

Landing page institucional do espaço de lazer Ele&amp;Ela.

## Tecnologias

- React 19 + Vite 8 + Tailwind CSS 4 + React Router 7
- Node.js + Express (backend)
- PostgreSQL

## Como rodar

```bash
npm install
npm run dev
```

Acessar em http://localhost:5173

## Scripts

| Comando           | Descrição                       |
|-------------------|---------------------------------|
| `npm run dev`     | API + Frontend simultâneos      |
| `npm run server`  | Apenas a API                    |
| `npm run build`   | Build de produção               |
| `npm run lint`    | Verificar código                |
